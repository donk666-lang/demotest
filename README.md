# Vue3.x使用G6绘制自适应图谱

## 一、需求

- 需求1：`Vue3.x`项目下使用`AntV G6`绘制图谱

- 需求2：图谱节点为两个`IP`地址，节点间存在**多条连线**情况
- 需求3：鼠标悬浮到节点上方时，高亮当前节点并出现`tooltip`气泡提示，展示相关信息

- 需求4：点击图谱连线后需要**高亮**关联节点和当前连线

- 需求5：图谱大小需要保证**自适应**（这里可以是根据窗口自适应，也可以是可调整大小的弹窗`DOM`内的自适应，后续会分开实现）
